# DrivAer CFD Validation

CFD investigation of Reynolds-number independence in automotive aerodynamics using full-scale and 30% scaled DrivAer notchback models.

## Overview

This project was completed during my internship at Tata Motors Passenger Vehicles Ltd.

The objective was to evaluate whether scaled automotive models can accurately reproduce full-scale aerodynamic behaviour for drag prediction and flow-field analysis.

The study compares a full-scale DrivAer notchback model against a 30% scaled geometry using ANSYS Fluent simulations and validates results against available wind-tunnel measurements.

## Software Used

- ANSA
- ANSYS Fluent
- EnSight

## Turbulence Model

- k-ω SST

## Key Results

| Parameter | Full Scale | 30% Scale |
|------------|------------|------------|
| Drag Coefficient (Cd) | 0.273 | 0.298 |
| Reynolds Number | 12.25 × 10⁶ | 4.25 × 10⁶ |

## Validation

- Wind Tunnel Cd = 0.277
- CFD Cd = 0.273
- Error = 1.44%
